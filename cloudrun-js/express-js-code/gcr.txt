2. Tag the local image with the remote repository 
    docker tag hello-express asia-south1-docker.pkg.dev/pristine-surf-404505/hello-express/hello-express:latest
3. push the image to the remote repository
    docker push asia-south1-docker.pkg.dev/pristine-surf-404505/hello-express/hello-express:latest

    1. login to gcp - gcloud auth login
